Strategy Spotlight: OCO Orders

OCO orders combine a limit order with a stop order:

  • Set profit target with limit order
  • Set stop loss simultaneously
  • When one triggers, other cancels automatically
// Example OCO Order Setup
{
    "symbol": "BTC/USDT",
    "side": "buy",
    "type": "oco",
    "price": 40000,     // Limit order price
    "stop_price": 39000, // Stop order trigger
    "quantity": 0.1
}
